# Чеклист разработчика

Проверки перед коммитом для уменьшения количества "фикс-коммитов".

> **Связанные документы:** [GLOSSARY.md](./GLOSSARY.md) • [KNOWN-ISSUES.md](./KNOWN-ISSUES.md)

---

## ⚡ Быстрая проверка (3 мин)

Минимум перед каждым коммитом:

- [ ] Код работает без ошибок в консоли
- [ ] Горячие клавиши работают (см. [GLOSSARY.md](./GLOSSARY.md#горячие-клавиши))
- [ ] Focus Session запускается и останавливается
- [ ] Блокировка сайтов активна во время сессии

---

## Полный чеклист

### Код

- [ ] Нет `console.log` (кроме отладки)
- [ ] Нет закомментированного кода
- [ ] Переменные и функции названы понятно
- [ ] Нет дублирования кода

### Функциональность

- [ ] Все кнопки кликабельны
- [ ] Навигация (назад/вперёд) работает
- [ ] Адресная строка принимает URL
- [ ] Страницы загружаются в webview

### Focus Mode

- [ ] Таймер отображается корректно
- [ ] Выбор времени (25/50/90) работает
- [ ] Блокировка активируется при старте
- [ ] Оверлей показывается на заблокированных сайтах

### UI

- [ ] Sidebar открывается/закрывается
- [ ] Insights сохраняются
- [ ] Нет визуальных глюков
- [ ] Адаптив работает (resize окна)

### Данные

- [ ] Данные сохраняются между перезапусками
- [ ] Формат данных соответствует ожидаемому
- [ ] Нет потери данных при ошибках

- [ ] Коммиты атомарные (одно изменение = один коммит)

- [ ] Ветка актуальна с main
