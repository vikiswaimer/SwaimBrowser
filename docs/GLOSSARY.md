# Глоссарий терминов Swaim Browser

Единая терминология для разработки и документации.

---

## Основные компоненты

| Термин | Определение | Где используется |
|--------|-------------|------------------|
| **Swaim Browser** | Специализированный браузер для продакт-менеджеров с фокусом на продуктивность | Везде |
| **Webview** | Элемент `<webview>` для отображения веб-страниц | `index.html` |
| **Sidebar** | Боковая панель с вкладками Focus/Projects/Insights | `index.html`, `.sidebar` |
| **Topbar** | Верхняя панель с URL-строкой и навигацией | `index.html`, `.topbar` |

---

## Focus Mode

| Термин | Определение | Код |
|--------|-------------|-----|
| **Focus Mode** | Режим глубокой фокусировки с таймером и блокировкой сайтов | `focusActive`, `toggleFocusMode()` |
| **Focus Overlay** | Полноэкранное затемнение во время Focus сессии | `#focusOverlay`, `.focus-overlay` |
| **Focus Timer** | Pomodoro-таймер (по умолчанию 25:00) | `focusTimeLeft`, `updateTimerDisplay()` |
| **Focus Goal** | Цель текущей сессии фокусировки | `#focusGoal`, `#sessionGoal` |
| **Session Duration** | Длительность сессии: 25, 50 или 90 минут | `#sessionDuration` |

---

## Блокировка сайтов

| Термин | Определение | Код |
|--------|-------------|-----|
| **Blocked Domains** | Список заблокированных доменов | `blockedDomains[]` |
| **Block Navigation** | Функция проверки и блокировки сайтов | `blockNavigation()` |

**Список заблокированных по умолчанию:**
- `youtube.com`
- `twitter.com`
- `reddit.com`
- `facebook.com`
- `instagram.com`

## Инсайты и данные

| Термин | Определение | Код |
|--------|-------------|-----|
| **Insight** | Записанная мысль/идея/боль с привязкой к URL | `store.get('insights')` |
| **Store** | Хранилище данных (electron-store) | `const store = new Store()` |

### Формат инсайта

```javascript
{
  url: string,       // URL страницы
  text: string,      // Текст инсайта
  timestamp: string, // ISO datetime
  tags: string[]     // Теги (пока не используются)
}
```

---

## Горячие клавиши

| Комбинация | Действие | Код |
|------------|----------|-----|
| **Alt+P** | Добавить инсайт с текущей страницы | `e.altKey && e.key === 'p'` |
| **Ctrl+Shift+F** | Включить/выключить Focus Mode | `e.ctrlKey && e.shiftKey && e.key === 'F'` |
| **Enter** (в URL-строке) | Перейти по URL или поиск | `urlInput keypress` |
