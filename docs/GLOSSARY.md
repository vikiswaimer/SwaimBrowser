# Глоссарий терминов Swaim

Единая терминология для использования в коде, коммитах и коммуникации.

## Deep Work

Состояние глубокой концентрации на когнитивно сложной задаче без отвлечений, позволяющее достичь максимальной продуктивности и качества работы.

## Product Framework

Встроенная система управления продуктовыми задачами и исследованиями, интегрированная в боковую панель браузера для организации работы над проектами.

## Insight

Ценная информация, идея или наблюдение, извлечённое из веб-страницы и сохранённое в системе для дальнейшего использования в продуктовых исследованиях.

## Focus Session

Ограниченный по времени период работы (25/50/90 минут) с активной блокировкой отвлекающих сайтов и таймером для поддержания концентрации.

## Blocked Domains (Заблокированные домены)

Список веб-сайтов (YouTube, Twitter, Reddit, Facebook, Instagram и др.), доступ к которым автоматически ограничивается во время активной Focus Session.

## Webview

Встроенный компонент для отображения веб-страниц внутри приложения. В NW.js использует тег `<webview>`.

## NW.js

Платформа для создания desktop-приложений на базе Chromium и Node.js. Альтернатива Electron.

## Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl/Cmd + L` | Фокус на адресную строку |
| `Ctrl/Cmd + R` | Перезагрузка страницы |
| `Ctrl/Cmd + Shift + I` | DevTools |
