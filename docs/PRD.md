## 1. Видение продукта

**Swaim Browser** — специализированный браузер для предпринимателей и продакт-менеджеров, работающих по методологии Lean Startup. Браузер объединяет глубокую фокусировку (Deep Work) со структурированным сбором инсайтов и продуктовым фреймворком.

### Миссия

Помочь digital nomads и стартаперам работать осознанно: не просто «сёрфить интернет», а систематически исследовать, фиксировать находки и превращать их в проверяемые гипотезы.

### Уникальное ценностное предложение (UVP)

> Единственный браузер, который совмещает **Focus Mode** для глубокой работы, **Product Framework** для структурирования гипотез, и **Insights** для захвата идей — всё в одном окне.

## 2. Целевой пользователь

### 2.1 Первичная аудитория: Digital Nomad + Lean Startup

| Характеристика | Описание |
|----------------|----------|
| **Кто** | Предприниматель-одиночка или маленькая команда (1-5 человек) |
| **Где работает** | Удалённо: кофейни, коворкинги, дом, путешествия |
| **Как работает** | Lean Startup: гипотезы → эксперименты → learning |
| **Боль #1** | Постоянные отвлечения (соцсети, YouTube) убивают глубокую работу |
| **Боль #2** | Инсайты из исследований теряются в хаосе вкладок и заметок |
| **Боль #3** | Нет единой системы для связи исследований с продуктовыми решениями |

### 2.2 Персона: Алекс

**Алекс, 32 года, digital nomad, строит SaaS-продукт**

- Живёт в Лиссабоне (сейчас), работает из коворкинга
- Запустил MVP, проводит customer discovery
- Каждый день: 50% исследования, 30% общение с клиентами, 20% разработка
- Использует: Notion (хаотично), Chrome (100+ вкладок), Pomodoro-приложение
- Проблема: открывает Twitter «на минутку» — теряет 40 минут

**Цитата:**
> «Мне нужен браузер, который поможет мне НЕ отвлекаться и сразу записывать найденное в нужное место.»

### 2.3 Jobs to Be Done

| Job | Описание | Частота |
|-----|----------|---------|
| **Deep Work Session** | Сфокусироваться на задаче без отвлечений | 3-5 раз в день |
| **Capture Insight** | Быстро записать мысль/боль/идею с контекстом | 10-20 раз в день |
| **Review & Connect** | Просмотреть инсайты, связать с гипотезами | 1-2 раза в день |
| **Block Distractions** | Не попасть на отвлекающие сайты | Постоянно |

## 3. Цели продукта

### 3.1 Бизнес-цели v0.1

| Цель | Метрика | Таргет |
|------|---------|--------|
| Валидация концепции | # активных пользователей | 100 DAU |

### 3.2 Пользовательские цели

1. **Глубокая концентрация** — провести рабочую сессию без отвлечений
2. **Структурированный сбор** — не терять инсайты из исследований
3. **Связь с продуктом** — видеть как инсайты влияют на решения

### 3.3 Ключевые метрики (North Star)

**Primary:** Количество завершённых Focus-сессий с сохранёнными инсайтами

**Supporting:**
- Время в Focus Mode / общее время
- Количество сохранённых инсайтов
- % сессий с заполненной целью

## 4. Scope v0.1

### 4.1 Deep Work Sessions (Focus Mode)

Ядро браузера — режим глубокой фокусировки.

#### Функциональность

| Функция | Описание | Приоритет |
|---------|----------|-----------|
| **Pomodoro Timer** | Таймер на 25/50/90 минут | P0 |
| **Session Goal** | Поле для цели сессии | P0 |
| **Full-screen Overlay** | Затемнение с таймером и целью | P0 |
| **Site Blocking** | Блокировка отвлекающих сайтов | P0 |
| **Visual Feedback** | Отображение заблокированного сайта | P1 |
| **Session History** | Список завершённых сессий | P1 |

#### Заблокированные сайты (по умолчанию)

- `youtube.com`
- `twitter.com` / `x.com`
- `reddit.com`
- `facebook.com`
- `instagram.com`
- `tiktok.com`

#### Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl+Shift+F` | Включить/выключить Focus Mode |
| `Esc` | Выйти из Focus Mode (с подтверждением) |

#### User Flow

```
1. Нажать ⚡ или Ctrl+Shift+F
2. Выбрать длительность (25/50/90 мин)
3. Ввести цель сессии
4. Работать (блокировка активна)
5. Таймер завершается → уведомление
6. Записать итоги (опционально)
```

### 4.2 Product Framework (Sidebar)

Боковая панель с продуктовым фреймворком Swaim.

#### Горячие клавиши

|------------|----------|
| `Ctrl+B` | Показать/скрыть Sidebar |

### 4.3 Insights System

Захват и хранение идей, болей, находок.

#### Функциональность

| Функция | Описание | Приоритет |
|---------|----------|-----------|
| **Quick Capture** | Сохранить инсайт с текущей страницы | P0 |
| **URL Binding** | Автоматическая привязка к URL | P0 |
| **Timestamp** | Время создания | P0 |
| **Inline Form** | Форма ввода без блокирующего prompt | P1 |
| **Tags** | Категоризация инсайтов | P2 |

#### Формат инсайта

```javascript
{
  id: string,           // UUID
  text: string,         // Текст инсайта (до 500 символов)
  url: string,          // URL страницы-источника
  timestamp: ISO8601,   // Время создания
  projectId?: string,   // ID связанного проекта
  tags: string[]        // Теги (v0.1: пустой массив)
}
```

#### Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Alt+P` | Открыть форму добавления инсайта |
| `Ctrl+Enter` | Сохранить инсайт |
| `Esc` | Отменить ввод |

#### User Flow

```
1. Находясь на странице, нажать Alt+P
2. Появляется inline-форма (не блокирующая)
3. Ввести инсайт
4. Ctrl+Enter → сохранено
5. Инсайт появляется в Sidebar
```

- Активная вкладка: подсветка акцентным цветом

- Позиция: верхний правый угол viewport

- Полноэкранное затемнение (98% opacity)
- Крупный таймер (48px, моноширинный)
- Цель сессии под таймером
- Уведомление о заблокированном сайте

## 7. Out of Scope (НЕ в v0.1)

### 7.1 Функции, которые НЕ делаем

| Функция | Причина | Планируется |
|---------|---------|-------------|
| **AI-ассистент** | Сложность, требует backend | v0.3 |
| **Knowledge Graph** | Требует больше данных | v0.3 |
| **Team Collaboration** | Фокус на одиночных пользователях | v0.4 |
| **Mobile App** | Сначала валидация на desktop | v0.5 |
| **Sync между устройствами** | Требует backend | v0.3 |
| **Gamification / Streaks** | Фокус на core функционал | v0.2 |
| **Интеграции (Notion, Linear)** | Фокус на standalone | v0.2 |
| **Кастомизация blocked sites** | Hardcoded достаточно для MVP | v0.2 |
| **Экспорт данных** | Не критично для MVP | v0.2 |
| **Browser Extensions** | Один продукт | Не планируется |
| **Криптовалюта / Rewards** | Не соответствует миссии | Не планируется |

### 7.2 Платформы, которые НЕ поддерживаем

- iOS / Android (mobile)
- Linux ARM (Raspberry Pi)
- Chrome/Firefox как расширение

### 7.3 Ограничения MVP

- Только локальное хранение данных
- Только Google для поиска
- Только один профиль пользователя
- Только один активный проект
- Нет истории браузера (только текущая сессия)
- Нет закладок

## 8. Приёмка и успех

### 8.1 Критерии готовности v0.1

| # | Критерий | Как проверить |
|---|----------|---------------|
| 1 | Focus Mode запускается и работает | Таймер тикает, overlay показывается |
| 2 | Сайты блокируются | Переход на youtube.com показывает блок |
| 3 | Инсайты сохраняются | Alt+P → ввод → отображается в Sidebar |
| 4 | Навигация работает | URL → Enter → страница загружается |
| 5 | Tabs работают | Ctrl+T создаёт, Ctrl+W закрывает |
| 6 | Данные персистятся | После перезапуска инсайты на месте |

### 8.2 Definition of Done

- [ ] Код работает без ошибок в консоли
- [ ] Все критерии готовности пройдены
- [ ] Документация обновлена (GLOSSARY, KNOWN-ISSUES)
- [ ] Проведён smoke test на Windows/macOS

### 8.3 Success Metrics (через 30 дней)

|---------|---------|------|
