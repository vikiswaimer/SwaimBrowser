# Swaim Browser

Браузер для фаундеров: **Deep Work** + **Product Discovery Framework**.

Swaim помогает основателям сфокусироваться на работе и системно собирать инсайты во время исследования рынка.

## Функции

### Deep Work (Focus Mode)
- Таймер сессий фокусировки (25 / 50 / 90 минут)
- Блокировка отвлекающих сайтов (YouTube, Twitter, Reddit, Facebook, Instagram)
- Установка цели на сессию
- История сессий

### Product Framework
- Сайдбар с вкладками для работы над продуктом
- Управление проектами
- Быстрое добавление инсайтов с текущей страницы

### Insights
- Сбор инсайтов, болей и идей во время серфинга
- Привязка к URL источника
- Тегирование записей
- История последних находок

## Установка и запуск

### Требования
- Node.js >= 18
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm start
```

## Сборка

### Собрать для всех платформ

```bash
npm run build
```

### Собрать для конкретной платформы

```bash
npm run build:linux   # Linux (x64)
npm run build:mac     # macOS (x64)
npm run build:win     # Windows (x64)
```

### Путь к бинарникам

После сборки бинарники находятся в папке `dist/`:

```
dist/
├── swaim-browser-linux-x64/
├── swaim-browser-osx-x64/
└── swaim-browser-win-x64/
```

## Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl+Shift+F` | Включить/выключить режим фокусировки (Focus Mode) |
| `Alt+P` | Добавить инсайт с текущей страницы |
| `Enter` (в адресной строке) | Перейти по URL или выполнить поиск в Google |
